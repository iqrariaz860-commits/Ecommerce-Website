

A dynamic, responsive web application built with Django to help small bakeries manage products and interact with customers online. 

✨ Key Features
Dynamic Product Display:Automatically renders the latest cupcake flavors and prices from the database. 
Customer Reviews:A dedicated section for clients to share feedback and ratings. 
Contact Management: Integrated contact form that saves inquiries directly to the database. 
Discount System:Dynamically calculates and displays discounted prices for promotional items.
Secure Architecture: Built-in protection against SQL injection and CSRF attacks. 

⚙️ Tech Stack
Backend: Python & Django (MVT Architecture)
Frontend: HTML, CSS, and JavaScript 
Database: SQLite (Development) 

🛠️ Future Roadmap
Implementation of a full user authentication system.
Addition of a shopping cart and online payment gateway.
Mobile application version.
